1. Отображение списка веток:
git branch: Эта команда отобразит список всех локальных веток в вашем репозитории. Текущая активная ветка будет выделена звездочкой (*).
Создание новой ветки:

2. git branch <branch_name>: Создает новую ветку с указанным именем, но не переключает вас на нее. Например, git branch feature-branch создаст новую ветку с именем "feature-branch".

3. Переключение на существующую ветку:
git checkout <branch_name>: Переключает ваш рабочий каталог на указанную ветку. Например, git checkout feature-branch переключит вас на ветку "feature-branch".

4. Создание и переключение на новую ветку:
git checkout -b <branch_name>: Создает новую ветку с указанным именем и сразу переключает вас на нее. Это объединяет функциональность git branch и git checkout в одной команде.

5. Удаление ветки:
git branch -d <branch_name>: Удаляет указанную ветку, если она была уже слита с текущей активной веткой.
git branch -D <branch_name>: Принудительно удаляет указанную ветку, даже если она не была слита с текущей активной веткой.

6. Отображение удаленных веток:
git branch -r: Показывает список удаленных (удаленных на удаленных серверах) веток.
Отображение всех веток (включая удаленные):

7. git branch -a: Отображает список всех веток, включая и локальные, и удаленные.
Поиск веток по шаблону:

8. git branch --list <pattern>: Отображает список веток, соответствующих указанному шаблону. Например, git branch --list feature-* выведет все ветки, начинающиеся с "feature-".
