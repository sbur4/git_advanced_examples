1. Добавление отдельного файла в индекс:
git add filename / git add *.txt 

2. Добавление всех измененных и новых файлов в индекс:
git add .

3. Добавление всех измененных и новых файлов в индекс в текущем каталоге:
git add -A
Это добавит все измененные и новые файлы в текущем каталоге и его подкаталогах в индекс. Этот вариант также учитывает удаленные файлы.

4. Добавление измененных файлов и удаленных файлов, но не новых файлов:
git add -u
Это добавит измененные файлы и удаленные файлы (но не новые файлы) в индекс.

5. Интерактивное добавление изменений:
git add -i
Это откроет интерактивный режим, который позволяет выбирать изменения для добавления в индекс.

6. Добавление только части изменений в файле (интерактивно) 
git add -p filename
Это откроет интерактивный режим, который позволяет выбирать части изменений в указанном файле для добавления в индекс.

7. Добавление файла с измененным именем (переименование):
git add --all
Если вы переименовали файл в рабочем каталоге, этот вариант автоматически обнаружит переименование и добавит его в индекс.

8. Добавление файлов с использованием шаблона (глоббинг):
git add "*.txt"
Это добавит все файлы, соответствующие указанному шаблону, в индекс.

Пожалуйста, помните, что после выполнения git add, вы все равно должны выполнить git commit, чтобы зафиксировать изменения в коммите.
