1. Простой вывод всех коммитов:
git log

2. Ограничение вывода на определенное количество последних коммитов (например, последние 5):
git log -n 5

3. Вывод краткой информации о коммитах (хеш, автор, дата, сообщение):
git log --oneline
git log --oneline -5(last five)

4. Вывод графа истории коммитов с визуализацией веток и слияний:
git log --graph
git log --graph --oneline
git log --graph --all --oneline -- decorate

5. Вывод измененных файлов в каждом коммите:
git log --stat

6. Вывод коммитов, сделанных в определенном диапазоне дат:
git log --since="2 weeks ago" --until="1 week ago"

7. Вывод коммитов, сделанных определенным автором:
git log --author="John Doe"

8. Вывод коммитов, связанных с определенным файлом или каталогом:
git log -- path/to/file_or_directory

9. Вывод коммитов, содержащих определенное ключевое слово в сообщении коммита:
perl
git log --grep="keyword"

10. Вывод коммитов с дополнительной информацией, такой как изменения в файлах и разница между коммитами:
git log -p

11. Вывод коммитов в определенном порядке сортировки (например, по дате создания):
git log --sort=committerdate

12. Вывод коммитов для конкретной ветки (например, main):
git log main

13. Вывод коммитов в виде одной строки с коротким хешем и сообщением:
git log --pretty=format:"%h - %s"

14. Вывод графа истории коммитов для конкретной ветки:
git log --graph main
