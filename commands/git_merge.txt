1. Объединение ветки в текущую ветку:
git merge <имя_ветки>
Эта команда объединяет указанную ветку (<имя_ветки>) в текущую ветку. Git попытается автоматически объединить изменения из выбранной ветки в текущую.

2. Fast-Forward объединение:
Если Git определяет, что нет конфликтов между текущей веткой и веткой, которую вы пытаетесь объединить, и история изменений линейна (нет веток), то происходит Fast-Forward объединение. В этом случае можно использовать следующую команду, чтобы объединить ветку:
git merge --ff-only <имя_ветки>

3. Объединение с автоматическим созданием коммита слияния:
Если есть конфликты между ветками, Git автоматически создаст коммит слияния. Вы можете использовать:
git merge --no-ff <имя_ветки>

чтобы гарантировать создание коммита слияния даже в случае возможного Fast-Forward объединения.

4. Отмена объединения:
Если вам нужно отменить недавнее объединение из-за конфликтов или других проблем, вы можете использовать команду:
git merge --abort
Эта команда отменяет текущий процесс объединения и возвращает вас в предыдущее состояние.

5. Объединение с удалением истории:
В редких случаях, когда вы хотите объединить ветку, но не сохранять ее историю, вы можете использовать опцию --squash:
git merge --squash <имя_ветки>

Это создаст один коммит, содержащий все изменения из выбранной ветки, но без сохранения ее истории.
