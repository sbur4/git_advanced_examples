1. Просто выполнение git status:
git status
Это выведет текущее состояние вашей ветки и информацию о неотслеживаемых файлах, измененных файлах и файлах, которые были добавлены в индекс, но не зафиксированы.

2. Использование флага -s или --short:
git status -s
Выводит краткую версию статуса, где каждая строка представляет собой сокращенное состояние файла в формате XY файл, где X - состояние индекса, а Y - состояние рабочего каталога.

3. Использование флага -b или --branch:
git status -b
Выводит дополнительную информацию о текущей ветке, включая имя текущей ветки и информацию о связи с удаленной веткой.

4. Использование флага -u или --untracked-files с разными параметрами:
git status -u
Выводит информацию о неотслеживаемых файлах. Может быть комбинирован с параметрами:

-no (по умолчанию): Показывает только неотслеживаемые файлы и игнорирует файлы, проигнорированные в .gitignore.
-normal: Показывает как неотслеживаемые файлы, так и игнорируемые файлы.
-all: Показывает все файлы, включая игнорируемые.

5. Использование других флагов и параметров для более детальной настройки вывода. Например, вы можете использовать --ignored для отображения игнорируемых файлов, или --porcelain для машинопонимаемого формата вывода.
