1. Просмотр информации о последнем коммите:
git show
Этот вариант команды git show покажет информацию о последнем коммите в текущей ветке, включая метаданные коммита (автор, дата), сообщение коммита и изменения, сделанные в этом коммите.

2. Просмотр информации о конкретном коммите:
git show <коммит-хеш>
Замените <коммит-хеш> на фактический хеш (SHA-1) коммита. Этот вариант команды git show покажет информацию о конкретном коммите, включая метаданные, сообщение коммита и изменения.

3. Просмотр изменений в определенном файле в коммите:
git show <коммит-хеш> <путь-к-файлу>
Этот вариант команды git show позволяет просмотреть изменения, сделанные в определенном файле в конкретном коммите. Вы укажете <коммит-хеш> и <путь-к-файлу> для этой цели.

Примеры:

git show abc12345 - Покажет информацию о коммите с хешем abc12345.
git show abc12345 path/to/file.txt - Покажет изменения в файле path/to/file.txt в коммите с хешем abc12345.

git show также может использоваться с другими параметрами для более специфического просмотра данных о коммите, но вышеупомянутые варианты являются наиболее распространенными.
